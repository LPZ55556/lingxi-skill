## 填充规则

1. 所有记忆必须基于原材料或用户口述，不得虚构
2. 时间信息尽量精确（从消息时间戳推断）
3. 地点信息可从照片 EXIF 或聊天内容中提取
4. 好的记忆和不好的记忆同等重要——都是真实的ta
5. 如果信息不足，标注 `[待补充]` 而非推测
6. 根据关系类型调整内容重点：
   - 恋人 → 约会记忆、甜蜜瞬间、争吵档案
   - 朋友 → 共同活动、玩笑互动、支持时刻
   - 家人 → 日常关怀、家庭活动、代际互动
   - 同事 → 工作配合、职场互动、私下交流
